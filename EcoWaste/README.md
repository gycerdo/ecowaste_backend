# EcoWaste 🌿
## Civic Intelligence App

### Setup
```bash
# 1. Install dependencies
flutter pub get

# 2. Update API URL in lib/services/api_service.dart
#    Change: static const String _baseUrl = 'https://YOUR-SERVER/api';

# 3. Run
flutter run
```
